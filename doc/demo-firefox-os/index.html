MathML.js is a Open Source (GLPv3) Javascript library (see the source code here and the documentation here)to enable GRAPHIC humam manipulation of MathML elements. This work is based on

  • TouchOfMath and
  • Equation Exercise Demo (proof of concept).

MathML.js ISN'T:

  • a plataform for display of mathematics in all browsers (try MathJax),
  • a equation editor (try Mathway),
  • a jquery-like Javascript library for MathML (try jquery-mathml);
  • a math Javascript library (try Math.js);
  • a symbolic math Javascript library (try Symbolic-Math-Library-for-Javascript).
